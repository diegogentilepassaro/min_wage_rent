OVERVIEW
==============================================================
The dataset contains the estimated count (in thousands) of rental houses for types making up the "single family / Condo Coop" Zillow category, broken down by number of bedrooms in the house. 

Geographic level: national 
Time level:       years 2011, 2013, 2015, 2017, 2019

SOURCE
==============================================================
Data manually compiled from https://www.census.gov/programs-surveys/ahs/data/interactive/ahstablecreator.html?s_areas=00000&s_year=2019&s_tablename=TABLE1&s_bygroup1=1&s_bygroup2=1&s_filtergroup1=1&s_filtergroup2=1

Filters: 

	- select area: national
	- select year: 2011, 2013, 2015, 2017, 2019
	- select a table: general housing
	- variable 1: number of bedroom 
	- tenure filter: renter

More information on: https://www.census.gov/programs-surveys/ahs/tech-documentation/user-notes/ahs-table-creator-release-notes.html

OVERVIEW
========================================================
The folder contains data from the US census and from the US ACS. Data are organized in subfolders accoring to geograhic level. 


SOURCE
========================================================
https://www.nhgis.org/ 			              Downloaded in multiple instances by Gabriele N Borg. 
                                         


DESCRIPTION
========================================================
ROOT/drive/raw_data/census		               Complete description

ROOT/drive/raw_data/census/block_group	               Block Group level Census. Different versions may contains different variables. Please refer to each version's codebook for the detailed description of the variables included. 

ROOT/drive/raw_data/census/block_group	               Tract Group level Census. Different versions may contains different variables. Please refer to each version's codebook for the detailed description of the variables included.
